## 🔬 Research and featured work

### Multimodal Gas Detection Benchmarking
*Transformer Architectures, Fusion Strategies, Experimental Benchmarking*

Built a multimodal model that fuses a **DeiT vision transformer** (thermal images) with an **LSTM** (gas sensor time series) to catch fire hazards earlier than any single sensor could. The fused model hit **100% test accuracy**, beating the prior benchmark of 96%, and I pushed the LSTM unimodal model from 82% to 91% using transfer learning. Explored both Early Fusion and Late Fusion strategies to balance how much each modality contributes.

### IoT and ML based Optimized Smart Irrigation System 📄 *Published in IEEE Xplore*
*Ensemble Methods, Stacking, Predictive Modeling*

Designed an end to end ML pipeline on real time IoT sensor data (temperature, humidity, soil moisture) streamed from the cloud, then built a **stacking ensemble** (SVM, MLP, Random Forest, AdaBoost, Gradient Boosting, XGBoost) that improved accuracy by **20%**. The system retrains itself periodically to keep up with changing field conditions. This work was presented at the **2023 ICSSIT International Conference** and indexed in **IEEE Xplore**.

### Jarvis, AI Desktop Assistant
*OpenCV, Computer Vision, Speech, Multithreading*

A Windows voice assistant with **face recognition authentication** and a vision based intrusion detection system that logs unauthorized access in real time. Added speech recognition, text to speech, and multithreaded background services for hands free, OS level automation.

### Real Time ID Tag Detection on Campus
*YOLO, Drone Footage, Dataset Annotation*

Led a team of 5 to capture and label **1,000+ drone images**, then trained a **YOLO** model for live ID tag detection on drone footage. Deployed it for continuous campus monitoring and improved security response efficiency by around 30%.
